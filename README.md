# DonutStats Player Search Extension

A small Chrome/Edge extension for searching DonutSMP player names from the toolbar. It fetches the public DonutStats profile page, shows the main player stats in the popup, remembers recent searches, and links to the full profile and vouch page.

## Install locally

1. Open `chrome://extensions` or `edge://extensions`.
2. Turn on `Developer mode`.
3. Choose `Load unpacked`.
4. Select this folder: `C:\Users\aaron\Documents\projects\donutstatsextension`.

## Use

Click the extension icon, type a Minecraft username, and press search. The extension shows a quick stats preview and keeps recent names for fast lookup.

## Notes

This is an unofficial extension and is not affiliated with DonutStats, DonutSMP, Mojang, or Microsoft. It uses public DonutStats pages from `https://donutstats.org/`.
